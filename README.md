# PERICORE TechStack

Interactive technology stack registry for the PERICORE multi-agent system.

**Live:** `https://<user>.github.io/pericore-techstack/`

## Setup

```bash
git clone https://github.com/<user>/pericore-techstack.git
cd pericore-techstack
npm install
npm run dev
```

## Deploy

Push to `main` auto-deploys via GitHub Actions to GitHub Pages.

## PWA

Open the live URL on iPhone/iPad, tap Share → "Add to Home Screen".
